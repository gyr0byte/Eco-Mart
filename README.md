## ğŸ“¥ Installation & Setup

Follow these steps to run Eco-Mart locally on your machine:

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A code editor (VS Code, Sublime Text, etc.) - optional
- Basic knowledge of HTML, CSS, and JavaScript

### Steps

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/eco-mart.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd eco-mart
   ```

3. **Open the project**

   Option 1: Open with Live Server (Recommended)
   - Install the "Live Server" extension in VS Code
   - Right-click on `HTML/index.html`
   - Select "Open with Live Server"

   Option 2: Open directly in browser
   - Navigate to the `HTML` folder
   - Double-click `index.html` to open in your default browser

4. **Start exploring!**
   - The homepage will load automatically
   - Navigate through different pages using the navbar

## ğŸš€ How to Use

### For Users

1. **Browse Products**
   - Visit the homepage and explore featured products
   - Navigate to the Products page to see the full catalog
   - Use filters to find specific eco-friendly items

2. **View Product Details**
   - Click on any product to view detailed information
   - Check sustainability ratings and product descriptions
   - See pricing and availability

3. **Shopping Cart**
   - Add products to your cart
   - Adjust quantities or remove items
   - Proceed to checkout (demo functionality)

4. **Explore Blog & Community**
   - Read articles about sustainable living
   - Check out community posts and tips
   - Learn about eco-friendly practices

### For Developers

1. **Modify Product Data**
   - Edit `JS/productData.js` to add/update products
   - Update product images in the `images/` folder

2. **Customize Styles**
   - Modify CSS files for different styling
   - All styles are modular and page-specific

3. **Add New Features**
   - JavaScript files are organized by functionality
   - Follow existing patterns for consistency
